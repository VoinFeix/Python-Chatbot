# Mini AI Agent - Chatbot 

This is a simple desktop chatbot app built with Python, Tkinter, and an AI API.

---
## Features

- Dark/light theme toggle
- Scrollable chat area
- Sends user messages to AI API and shows responses
- Error handling for API failures
- Clear chat and exit buttons

---
## Requirements

- Python 3.x
- `requests` library (`pip install requests`)

---
## Setup

1. Get your **API key** from your Groq cloud at https://console.groq.com/.

2. Replace the `API_KEY` variable in `main.py` with your actual API key:

```python
API_KEY = "your_api_key_here"
```

---
## Instalation
1. Clone the Repository
```bash
git clone https://github.com/DevManoj96/Python-Chatbot.git
cd Python-Chatbot
```
2. Run the program
```bash
python3 main.py
```

---
## Usages
- Type your message and press Enter or click Send.
- Toggle between dark and light mode using the Toggle Theme button.
- Clear chat history with Clear Chat button.
- Exit the app with Exit button or by typing "exit", "quit", or "bye".

---
## Note
- This app depends on a working internet connection and a valid API key.
